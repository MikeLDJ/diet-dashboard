# diet-dashboard

## Coding task for GR

### Require:

- NodeJS

- gulp
```
npm install -g gulp
```
or
```
sudo npm install -g gulp
```
***
### Install:

```
npm install
```

***
## Use:

### development:

```
gulp serve
```

open browser on ```localhost:3100```

files in ```app``` directiory

### production:

```
gulp
```
files in ```dist``` directiory
